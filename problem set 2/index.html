Problem Set 2: The Grid

  • Click a square to highlight it
  • How it works
    • Use the arrow keys on the keyboard to move to a different square
    • If "Enable Drawing" is checked, clicking or arrowing to a square should turn black.
    • If "Enable Drawing" is not checked, clicking or arrowing to a square should not change the square color.
    • Click "Erase Drawing" to remove the black from every square.
